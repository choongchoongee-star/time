# 시간 가계부 (Time Ledger)

하루 시간을 어디에 썼는지 기록하는 개인용 웹앱. 서버·로그인 없이 브라우저(localStorage)에 저장합니다.

## 기능
- 카테고리 + 시작/종료 시각으로 활동 기록 (소요 시간 자동 계산, 메모 한 줄)
- 날짜별 기록 목록과 하루 총합 / 남은 시간
- 카테고리별 요약 (오늘 / 이번 주) — 비율 막대
- 카테고리 추가·삭제(색상 지정)
- JSON 백업 내보내기 / 불러오기
- PWA: iOS Safari에서 "홈 화면에 추가" 시 앱처럼 전체화면 사용, 오프라인 동작

## 사용 (GitHub Pages)
1. 이 저장소의 **Settings → Pages** 에서 Source를 `main` 브랜치 / `/ (root)` 로 설정
2. 잠시 후 `https://choongchoongee-star.github.io/time/` 접속
3. 아이폰 Safari에서 열고 **공유 → 홈 화면에 추가**

## 데이터 보관 주의
기록은 **그 브라우저에만** 저장됩니다. 캐시/사이트 데이터 삭제 시 사라지므로,
가끔 **백업 내보내기**로 JSON 파일을 보관하세요.

## 파일 구성
- `index.html` — 앱 전체 (HTML/CSS/JS 한 파일)
- `manifest.json` — PWA 매니페스트
- `sw.js` — 오프라인 캐시 서비스워커
- `icon-*.png` — 앱 아이콘
